# Jewelkunst Landing Page

Eine hochwertige, statische Landingpage fuer **Jewelkunst** auf GitHub Pages.

## Inhalt

- Premium-Startseite in HTML/CSS
- Mobile-optimiertes Layout
- SEO-Titel und Meta Description
- GitHub-Pages-Workflow fuer automatisches Deployment
- `.nojekyll` fuer saubere statische Auslieferung

## Live URL

Sobald GitHub Pages im Repository aktiviert ist:

```text
https://nicofrommer-web.github.io/Projekt/
```

## Deployment

Der Deploy laeuft ueber GitHub Actions:

```text
.github/workflows/pages.yml
```

Wichtig in GitHub:

1. Repository oeffnen
2. Settings > Pages
3. Source auf **GitHub Actions** stellen
4. Workflow erneut starten

## Marke

Jewelkunst steht fuer ruhige, warme und bedeutungsvolle Schmuckpraesenz: Halsketten, Armbaender, Ohrringe und zeitlose Schmuckstuecke mit leiser Eleganz.
